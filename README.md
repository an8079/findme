# findme


## 安装

将 `commands/findme.md` 放到你的 opencode 配置目录：

```
~/.config/opencode/commands/findme.md
```

## 使用

在 opencode 中发送自然语言：

> > 下载 https://raw.githubusercontent.com/an8079/findme/main/commands/findme.md 这个文件并保存到 `~/.config/opencode/commands/findme.md`，


或直接输入 `/findme`（如果你的 opencode 支持 slash 命令）。
